# Nathafty Customer Survey

Application Next.js pour les enquêtes de satisfaction client Nathafty.

## 🚀 Installation

```bash
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## 📋 Fonctionnalités

- ✅ Page d'accueil vitrine Nathafty
- ✅ Formulaire d'enquête sur la gestion des déchets
- ✅ Visualisation des résultats avec Chart.js
- ✅ Design responsive avec Tailwind CSS
- ✅ Sauvegarde Supabase (PostgreSQL)

## 🎨 Technologies

- **Next.js 15** - Framework React
- **TypeScript** - Typage statique
- **Tailwind CSS** - Styling
- **Supabase** - Base de données
- **Chart.js** (via CDN) - Visualisation
